Evaluation

The Python framework contains the basic functionality that is used by the individual evaluation scripts to import and export data, as well as initialize and employ the machine-learning functionalities of the scikit-learn framework.

The evaluation scripts encode the individual experiments that were run on the evaluation server.

The evaluation results contain the results of individual experiments, encoded in csv, txt, and json file formats.

Tool

Fritz is an executable jar file of the tool that was used to compute the metric values for each of the input datasets. Additional Information regarding the requirements, functionalities, and usage of Fritz is available at the tool web page

Appendix

The online appendix provides additional material, referred to within the original paper.
It includes full-sized plots and descriptions of all results of studies 1 and 2:

  • Study 1: Individual metrics as fault predictors
  • Study 2: Fault prediction with multiple metrics

And also adds descriptions and results of additional experiments that were not part of the main studies:

  • Effect of limiting features on Random Forest classifiers
  • Most important features of Random Forest classifiers
  • Transfer learning between datasets
  • Combination of datasets for training
  • Combination of datasets for evaluation

A pdf appendix provides additinal information and explanations for studies one and two.
