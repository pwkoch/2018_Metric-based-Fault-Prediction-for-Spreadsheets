Evaluation

The Python framework contains the basic functionality that is used by the individual evaluation scripts to import and export data, as well as initialize and employ the machine-learning functionalities of the scikit-learn framework.

The evaluation scripts encode the individual experiments that were run on the evaluation server.

The evaluation results contain the results of individual experiments, encoded in csv, txt, and json file formats.

Tool

Fritz is an executable jar file of the tool that was used to compute the metric values for each of the input datasets. Additional Information regarding the requirements, functionalities, and usage of Fritz is available at the tool web page

Dataset Combination

Prediction performance of a learned classifier highly depends on the qulity and consistency of the utilized dataset. In the main studies of our work, we used our available datasets (Enron Errors, Info1, Euses) in separation. In this study, we hence evaluate the combinability of those datasets by evaluating different combination using a trained Random Forest classifier.
