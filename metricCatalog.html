Description

One main contribution of our work is a catalog of 64 metrics, based both on existing work from the literature and our own expertise. When selecting metrics for the catalog, our goal was to provide a broad range of different measurements to cover a wide variety of faults occurring in practice. We first scanned previous research that discussed possible characteristics for assessing the quality of spreadsheets and extracted those ideas that seemed likely to be correlated with certain types of faults in spreadsheets. Not all identified characteristics could be applied directly. In such cases, the general idea of the metric was adapted so that it can be measured in terms of a numeric value. For example, Metric 2, Column distance to relative, is conceptually based on the Pattern Finder spreadsheet smell. Likewise, except for certain spreadsheet smells, worksheet metrics are uncommon in the current literature. Many stated worksheet metrics are therefore designated as novel, however they often use concepts that are common for non-worksheet contexts like the number of formula cells in a spreadsheet. We present the catalog using a collection of three tables:

  • Table 1 provides an overview of the metric catalog.
  • Table 2 extends Table 1 with a more detailed description and explanation for each metric.
  • Table 3 provides a literature list that is referred to in Tables 1 and 2.

1 Column Cell Novel
Description Column number (position) of the cell in the worksheet.
Explanation The further away a column is from a worksheet's origin (column A), the less likely it is to be checked and the harder it is to verify its references. In addition, under the assumption that spreadsheets are developed from top to bottom and from left to right, we may also observe more inputs in the top left corner and more complex aggregation functions farther below and to the right.

2 Column distance to relative Cell 7,8
Description Distance to the next cell in the same column that has the same cell type (e.g. number or string); we use a predefined and configurable maximum value for this metric. Adapted from Pattern Finder to provide a numeric score.
Explanation Spreadsheets usually group collections of items (labels, input, calculations) into neighboring columns and/or rows of cells that consequently have the same type. Cells of a specific type far away from other cells of the same type might therefore hint at mistakes in such setups.
